Overview

This document summarizes the Computer Adaptive Testing (CAT) survey on problematic mobile phone use. Instrument definitions follow the project item bank (ITEM BANK_Problematic Mobile Phone CAT.docx). All tables and figures below are computed live from DATA_Problematic Mobile Phone Use CAT.csv.

Instrument reference

From the item bank documentation:

Code 
kable(scale_defs, align = c("l", "l", "r", "c", "l"))

Demographics: gender (1 = Male, 2 = Female), grade (see codebook), age.

Note: SPAI_12 is item 12 of the SPAI inventory (4-point response) and is also used as a categorical severity indicator in classification models.

Item-level means by scale

Average response per item (complete cases), faceted by instrument.

Code 
item_means_list <- lapply(seq_len(nrow(scale_defs)), function(i) {
  cols <- scale_item_cols(dat, scale_defs$Item_prefix[i])
  tibble(
    item = cols,
    mean_score = colMeans(dat[, cols], na.rm = TRUE),
    scale = scale_defs$Label[i]
  )
})
item_means_df <- bind_rows(item_means_list) %>%
  group_by(scale) %>%
  mutate(item = reorder(item, mean_score)) %>%
  ungroup()

ggplot(item_means_df, aes(x = mean_score, y = item, fill = mean_score)) +
  geom_col(show.legend = FALSE) +
  facet_wrap(~ scale, scales = "free_y", ncol = 1) +
  labs(x = "Mean item score", y = NULL, title = "Average endorsement by item") +
  theme_minimal(base_size = 9) +
  theme(strip.text = element_text(size = 8))

Summary

  • 1619 respondents in the raw file; 1619 complete cases used for all summaries above.
  • Seven instrument families plus demographics, documented in the CAT item bank.
  • Scale totals are strongly correlated with one another; SAPS, SPAI, MPAS, MPATS, and SAS tend to move together (see correlation heatmap).
  • Demographic breakdowns and item-level means help locate where respondents endorse problematic phone-use symptoms most strongly.
